IMPORTANT: You must change your default download location in Chrome to the network share connected to the printer for this workstation. Failure to do this step will result in this extension not doing anything!

In the settings window:

  1. Click 'Show advanced settings...'
  2. Scroll down to 'Downloads'
  3. Click the 'Change...' button
  4. Select the correct network share to save files to and click 'OK'

Finally, ensure the 'Ask where to save each file before downloading' is unchecked.
